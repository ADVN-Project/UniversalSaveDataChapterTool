# Universal SaveData Chapter Tool
"Universal SaveData Chapter Tool" 

是一个用于剪切 Savedata 文件夹内容到 Savedata.partXXX文件夹的工具，

实现存档位扩展功能，方便存档不够用的玩家加载以前的存档。
